# GLSL Parser
This is a GLSL parser written as a webpack loader, this loader will mangleyour GLSL code with the following modificitons.

Modifications 
------------- 
* Obfuscates private variables. 
* Removes comments.
* Removes whitespaces.

These instructions will help you .
